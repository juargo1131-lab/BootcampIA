# 📊 Proyecto de Análisis de Datos con Python

Este repositorio contiene un **proyecto académico desarrollado con fines educativos** en el marco de un **curso de Análisis de Datos**, utilizando **Python** y **Google Colab** como herramientas principales.

El proyecto tiene como finalidad aplicar de manera práctica los conceptos teóricos vistos en el curso, mediante el análisis, procesamiento e interpretación de datos.

---

## 🎯 Objetivo del proyecto

- Aplicar técnicas fundamentales de análisis de datos.
- Desarrollar habilidades en el uso de Python para el tratamiento de información.
- Realizar exploración, limpieza y visualización de datos.
- Interpretar resultados y comunicar hallazgos de forma clara y estructurada.

---

## 🧠 Contexto académico

Este proyecto está orientado exclusivamente a:
- Aprendizaje y práctica académica.
- Evaluación de competencias en análisis de datos.
- Reforzamiento de conocimientos técnicos.

No está diseñado para uso en entornos productivos.

---

## 🛠️ Tecnologías y herramientas utilizadas

- **Lenguaje de programación:** Python  
- **Entorno de trabajo:** Google Colab  
- **Librerías principales:**
  - `pandas`
  - `numpy`
  - `matplotlib`
  - `seaborn`

> Todas las herramientas utilizadas forman parte del ecosistema estándar de análisis de datos en Python.
